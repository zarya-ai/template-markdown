# Developer & Coding Templates

Templates for developers, DevOps engineers, and technical professionals working with infrastructure, scripts, and configurations.

## Templates in This Category

- **README.md** — This file
- **shell-script.md** — Bash/Shell script template with best practices
- **dockerfile.md** — Docker configuration template with multi-stage builds
- **docker-compose.md** — Docker Compose orchestration template
- **yaml-config.md** — YAML configuration file template
- **dotfiles-organization.md** — Guide for organizing dotfiles with best practices
- **script-consolidation.md** — Template for consolidating and reverse-engineering scripts
- **makefile.md** — Makefile template for automation
- **systemd-service.md** — Systemd service file template
- **network-config.md** — Network configuration template for home lab/servers
- **dev-environment-setup.md** — Development environment initialization script
- **code-snippet-reference.md** — Snippet library and code reference organization

## Usage

These templates are designed to help you:

1. **Consolidate** code across multiple versions
2. **Standardize** configuration patterns
3. **Reference** common snippets and best practices
4. **Organize** infrastructure code and scripts
5. **Document** technical configurations

## Quick Start

Use the `script-consolidation.md` template to reverse-engineer and merge multiple versions of your scripts and configurations. Then apply the appropriate category template for the final artifact.
