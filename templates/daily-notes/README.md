# Daily Notes Templates

Templates for organizing daily notes, journaling, and tracking various aspects of your life in Obsidian.

## Templates in This Category

- **README.md** — This file
- **daily-log.md** — Standard daily log/journal entry
- **daily-standup.md** — Daily standup for work/projects
- **daily-review.md** — End-of-day reflection and review
- **morning-pages.md** — Morning brain dump and planning
- **habit-tracker.md** — Daily habit tracking and monitoring
- **mood-and-energy.md** — Mood, energy, and wellness tracking
- **gratitude-journal.md** — Gratitude practice and appreciation
- **learning-log.md** — Daily learning and development log
- **workout-log.md** — Exercise and fitness tracking
- **meal-plan.md** — Daily meal planning and nutrition tracking
- **time-tracking.md** — Time blocking and activity tracking
- **reflection-prompt.md** — Guided reflection with prompts
- **three-good-things.md** — Three good things exercise
- **weekly-summary.md** — End-of-week summary and planning

## Usage

Create daily note files using these templates:

```bash
# In Obsidian, create new notes named with the date
2026-06-23 - Daily Log.md
2026-06-23 - Daily Standup.md
2026-06-23 - Habit Tracker.md
```

Or use Obsidian's daily notes feature with the "Daily Notes" plugin.

## Linking & Backlinks

Link related notes together:

```markdown
## Related
- [[2026-06-23 - Daily Log]]
- [[2026-06-22 - Daily Review]]
- [[Week 26 Summary]]
```

## Tags for Organization

Use consistent tags:

```markdown
#daily-log #2026-06 #reflection
#workout #morning #energy-high
#learning #development #python
```

## Quick Start

1. Choose a template from the list above
2. Copy the template content
3. Create a new daily note in your vault
4. Customize the template with today's date and information
5. Use tags and links to organize and connect notes
