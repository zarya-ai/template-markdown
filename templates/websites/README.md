# Website Templates

Templates for planning, documenting, and managing websites.

## Templates in This Category

- **site-plan.md** — Overall website structure and strategy
- **page-template.md** — Individual page documentation
- **seo-checklist.md** — SEO optimization checklist
- **content-calendar.md** — Editorial and content calendar

## Usage

Use these templates to plan website architecture, document pages, track SEO, and manage content publishing schedules.
