# Documentation Templates

A collection of templates for creating comprehensive documentation.

## Templates in This Category

- **guide.md** — Step-by-step how-to guide template
- **api-reference.md** — API documentation reference
- **faq.md** — Frequently asked questions format

## Usage

Each template provides a structured starting point for different documentation types. Copy the appropriate template to your vault and fill in the `[FILL]` sections with your content.
